# nix-shells

## nix-templates

collection of templates to work on various project types.

### ansible

```
nix flake init --template github:gfriloux/nix-shells#ansible
```

### mdBook

```
nix flake init --template github:gfriloux/nix-shells#mdbook
```

### terraform

```
nix flake init --template github:gfriloux/nix-shells#terraform
```
